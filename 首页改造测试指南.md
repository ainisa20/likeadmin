## 🐛 常见问题排查

### 问题 1: 页面显示空白或旧内容

**原因**: 浏览器缓存

**解决方案**:
1. 强制刷新：`Cmd+Shift+R` (Mac) 或 `Ctrl+Shift+R` (Windows)
2. 清除浏览器缓存
3. 尝试无痕模式访问

### 问题 2: 表单提交无响应

**原因**: 后端接口未对接

**解决方案**:
- 当前提交是模拟的，会显示成功提示
- 实际功能需要对接后端 API（待开发）

### 问题 3: 样式显示异常

**原因**: CSS 文件加载失败

**检查**:
```bash
# 检查 CSS 文件是否存在
ls server/public/pc/_nuxt/entry.*.css

# 测试 CSS 文件可访问性
curl -I http://127.0.0.1:8088/pc/_nuxt/entry.*.css
```

### 问题 4: 控制台有错误

**检查步骤**:
1. 打开浏览器开发者工具（F12）
2. 切换到 Console 标签
3. 截图错误信息
4. 提供错误堆栈
