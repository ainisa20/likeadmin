## 💡 使用建议

### 最佳实践

1. **粘贴前检查**
   - 如果从代码文件、JSON、配置文件中复制内容
   - 可能包含转义字符
   - 直接粘贴到对话框即可，系统会自动处理

2. **特殊字符处理**
   - 如果确实需要显示 `\n`（如教程）
   - 使用 `\\n` 表示一个反斜杠加n

3. **复杂内容分段**
   - 长文章建议分章节转换
   - 每次转换2000-3000字
   - 避免一次性处理过多内容

### 常见问题

**Q: 为什么还有 `\n` 显示？**
A:
1. 清除浏览器缓存并刷新
2. 确认已使用最新构建的前端代码
3. 检查文章是否需要重新编辑保存

**Q: 代码块中的 `\n` 也被转义了？**
A: 这是正常的。代码块中的换行会被保留，不会显示为字面的 `\n`。

**Q: 如何确认修复已生效？**
A:
1. 打开浏览器控制台（F12）
2. 切换到 Network 标签
3. 刷新页面
4. 查找 `index-xxx.js` 文件
5. 确认文件时间是最新的
