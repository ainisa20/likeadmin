# 经营范围规范表述查询 Skill

**访问国家市场监管总局的经营范围规范表述查询系统，获取官方认可的标准化条目。**

---

## 系统信息

**经营范围规范表述查询系统：**
- 系统名称：经营范围规范表述查询系统（试用版）2.5.15
- 官方网址：https://jyfwyun.com/
- 管理机构：国家市场监管总局
- 特点：✅ 可自动化访问，✅ 无需登录，✅ 无验证码

---

## 使用场景

当用户需要：
- 查询特定业务的经营范围规范表述
- 设计企业经营范围
- 确保经营范围表述符合工商注册要求
- 获取多方案经营范围设计对比

---

## 查询流程

### 第一步：访问系统

```
browser_use.open("https://jyfwyun.com/")
```

### 第二步：选择地区

**支持的地区：**
- 全国
- 各省、直辖市、自治区
- 各地级市

**选择建议：** 全国（适用于大多数企业）

### 第三步：查询关键词

**输入核心业务关键词：**
- 软件开发
- 餐饮服务
- 电子商务
- 建筑工程
- 进出口贸易

**查询特点：**
- 支持模糊搜索
- 返回多个相关条目
- 每个条目都是官方规范表述

### 第四步：获取规范表述

**系统返回的内容：**
- 规范化表述条目
- 条目代码
- 所属类别
- 相关条目

**示例：**
```
输入：软件开发
输出：
- 软件开发
- 信息系统集成服务
- 数据处理和存储支持服务
- 信息技术咨询服务
- 计算机系统服务
```

### 第五步：策略性组合设计

**策略架构：**

```
核心业务（立足当下）
    ↓
延伸业务（纵向拓展）
    ↓
辐射业务（横向拓展）
    ↓
通用业务（安全垫）
```

**详细说明：**

| 层级 | 定义 | 示例 |
|------|------|------|
| **核心业务** | 最主要、最盈利的业务 | 软件开发 |
| **延伸业务** | 产业链上下游 | 信息系统集成服务、软件销售 |
| **辐射业务** | 相关衍生领域，未来1-2年 | AI软件开发、互联网数据服务 |
| **通用业务** | 几乎所有公司都可能涉及的 | 企业管理咨询、会议展览 |

### 第六步：多方案对比

**方案A - 轻量化：**
- 条目数：3-5个
- 特点：简洁，成本低
- 适用：初创企业

**方案B - 标准型：**
- 条目数：5-10个
- 特点：平衡，适中
- 适用：成长期企业

**方案C - 扩展型：**
- 条目数：10-20个
- 特点：广泛，灵活
- 适用：快速扩张企业

## 使用注意事项

### 系统使用

1. **使用频率**
   - 避免短时间内大量查询
   - 建议每次查询间隔 2-3 秒

2. **地区差异**
   - 不同地区可能有细微差异
   - 建议选择注册地所在地区

3. **条目更新**
   - 系统会定期更新条目
   - 条目更新数量：4208+（截至2025年）

### 设计建议

1. **关键词选择**
   - 使用核心业务词汇
   - 避免过于宽泛或过于狭窄
   - 可以尝试多个相关关键词

2. **方案选择**
   - 初创企业：选择轻量化方案
   - 成长期企业：选择标准型方案
   - 快速扩张：选择扩展型方案

3. **后续调整**
   - 经营范围可以变更
   - 变更时间：3-5 个工作日
   - 费用：通常几百元

### 法律风险

**超范围经营风险：**
- 属于违法行为
- 最高罚款：10 万元
- 建议谨慎选择，覆盖实际业务

## 常见问题

### Q1：查询结果可以保证通过吗？

**A：** 查询结果是官方规范表述，可以显著提高通过率，但最终以工商局审核为准。建议准备 2-3 个备选方案。

### Q2：哪些业务需要前置审批？

**A：** 如医疗器械、金融服务、教育培训等，需要先取得许可证。本系统会标注是否需要许可。

### Q3：经营范围可以变更吗？

**A：** 可以。需要到工商局办理变更登记，通常需要 3-5 个工作日，费用几百元。

### Q4：如何选择合适的方案？

**A：**
- 初创企业：轻量化方案（节省成本）
- 成长期企业：标准型方案（平衡发展）
- 快速扩张：扩展型方案（灵活应对）

### Q5：系统更新频率如何？

**A：** 系统定期更新，已更新 4208+ 条次。建议查询时使用最新数据。
