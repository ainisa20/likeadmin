# Markdown 粘贴功能使用说明

## 功能概述

文章编辑器现已支持将 Markdown 格式的内容转换为 HTML，方便快速粘贴和格式化文章内容。

## 使用步骤

### 1. 打开文章编辑页面
- 进入后台管理系统
- 点击「文章管理」→「文章管理」
- 点击「添加」或编辑现有文章

### 2. 使用 Markdown 粘贴功能
- 在编辑器顶部找到「**粘贴Markdown**」按钮
- 点击按钮打开对话框
- 将 Markdown 内容粘贴到文本框中
- 点击「**转换为HTML**」按钮
- 内容会自动插入到编辑器中

## 支持的 Markdown 语法

### 文本格式
```markdown
**加粗文本**
*斜体文本*
***粗斜体文本***
~~删除线~~
```

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 列表
```markdown
- 无序列表项1
- 无序列表项2
  - 嵌套列表项

1. 有序列表项1
2. 有序列表项2
3. 有序列表项3
```

### 引用
```markdown
> 这是一段引用文本
> 可以有多行
```

### 链接和图片
```markdown
[链接文字](https://example.com)
![图片描述](https://example.com/image.jpg)
```

### 代码块
```markdown
`行内代码`

\`\`\`javascript
// 代码块
function hello() {
    console.log('Hello World');
}
\`\`\`
```

### 表格
```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

### 分割线
```markdown
---
```

## 常见问题

### Q1：转换后格式不对怎么办？
A：检查原始Markdown语法是否正确，确保符合标准Markdown格式规范。

### Q2：图片不显示？
A：确保图片URL是完整的http/https地址，或者使用系统的素材库功能上传图片。

### Q3：代码块没有语法高亮？
A：转换后的代码块会保留基本格式，如需语法高亮，可以在编辑器中手动设置代码块语言。

### Q4：表格显示不正常？
A：Markdown表格对格式要求严格，确保：
- 每行用 `|` 分隔
- 表头下方有 `---` 分隔线
- 列数对齐

## 注意事项

1. **内容审核**：转换后请预览效果，必要时手动调整格式
2. **图片处理**：建议使用系统素材库上传图片，而非外部链接
3. **长文档**：对于特别长的文档，建议分段转换，避免一次性处理过多内容
4. **格式保留**：转换后的内容可以在编辑器中继续编辑和调整

## 技术实现

- **Markdown解析**：使用 `marked` 库进行转换
- **配置选项**：
  - `breaks: true` - 支持单个换行符转换为 `<br>`
  - `gfm: true` - 启用GitHub Flavored Markdown
