# Markdown粘贴功能 - 快速开始

## 1分钟快速上手

### 第一步：打开文章编辑器
1. 登录后台
2. 文章资讯 → 文章管理 → 点击「添加」

### 第二步：粘贴内容
1. 点击编辑器顶部的 **「粘贴Markdown」** 按钮
2. 粘贴你的Markdown内容
3. 点击 **「转换为HTML」**

### 第三步：完成
✅ 内容已自动格式化并插入编辑器！

## 实例演示

### 你想粘贴这样的内容：

```markdown
# 公司新政策通知

## 重要变更

为了提供更好的服务，我们将于**下月1日**实施以下新政策：

### 主要变更

1. 延长保修期至2年
2. 提供免费技术支持
3. 优化退换货流程

### 特别说明

> 所有老客户自动享受新政策，无需额外申请。

如有疑问，请联系客服。

---

*感谢您的支持！*
```

### 转换后自动变成：

✅ 大标题居中
✅ 小标题层级分明
✅ "下月1日"加粗显示
✅ 列表自动编号
✅ 引用块有灰色背景
✅ 分隔线清晰
✅ 底部文字斜体

**无需手动调整格式！**

## 使用技巧

### 技巧1：从其他平台迁移
从知乎、简书、CSDN等平台复制的内容如果是Markdown格式，可以直接粘贴转换。

### 技巧2：使用模板
创建常用的文章模板，保存为Markdown格式，每次使用时直接粘贴。

### 技巧3：快速排版
不需要在编辑器里反复点击格式按钮，直接用Markdown语法一次性搞定。

### 技巧4：代码文章
写技术文章时，用Markdown的代码块语法，格式更规范。

## 常见问题速查

**Q: 点击按钮没反应？**
A: 刷新页面重试

**Q: 转换后格式不对？**
A: 检查Markdown语法是否正确

**Q: 想继续编辑怎么办？**
A: 转换后可以在编辑器中继续编辑

**Q: 支持图片吗？**
A: 支持Markdown图片语法，建议用素材库上传

**Q: 能粘贴很长的文章吗？**
A: 可以，但建议分段处理
