## 🔧 技术实现

### 组件使用

| 组件 | 用途 | 配置 |
|------|------|------|
| `ElButton` | CTA按钮 | `type="primary"`, `size="large"` |
| `ElForm` | 表单容器 | `label-position="top"` |
| `ElInput` | 文本输入 | `size="large"`, `maxlength="11"` |
| `ElSelect` | 下拉选择 | `size="large"`, 占满宽度 |
| `ElIcon` | 图标 | `InfoFilled` 提示图标 |

### 表单验证

```typescript
// 手机号正则验证
{ pattern: /^1[3-9]\d{9}$/, message: '请输入正确的手机号' }

// 必填项验证

```
